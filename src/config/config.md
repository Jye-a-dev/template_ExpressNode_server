# config

English: This folder stores application configuration such as environment variables, database setup placeholders, and Swagger options. Keep configuration centralized so other folders can import from one predictable place.

Tiếng Việt: Folder này lưu cấu hình của ứng dụng như biến môi trường, khung cấu hình database và tùy chọn Swagger. Nên gom cấu hình tại đây để các folder khác import từ một nơi rõ ràng.
