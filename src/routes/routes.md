# routes

English: This folder contains the root router that combines module routers and exposes them under the API prefix. Keep global route mounting here instead of scattering it through the app bootstrap.

Tiếng Việt: Folder này chứa router gốc dùng để gom router của các module và expose chúng dưới API prefix. Nên đặt việc mount route tổng tại đây thay vì rải trong phần bootstrap ứng dụng.
