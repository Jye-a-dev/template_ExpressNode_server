# modules

English: This folder contains feature modules. Each module should own its route, type, controller, model, validator, and service files so features stay isolated and easy to maintain.

Tiếng Việt: Folder này chứa các module tính năng. Mỗi module nên tự quản lý các file route, type, controller, model, validator và service để tính năng độc lập, dễ bảo trì.
