# template_modules

English: This is the example module folder. Copy or rename it when creating a real feature module, then fill in the six files according to the comments inside each file.

Tiếng Việt: Đây là folder module mẫu. Khi tạo module thật, hãy sao chép hoặc đổi tên folder này, sau đó điền nội dung vào 6 file theo phần comment hướng dẫn trong từng file.

Expected files:

- `template_modules.route.ts`: route declarations
- `template_modules.type.ts`: TypeScript types
- `template_modules.controller.ts`: HTTP controller functions
- `template_modules.model.ts`: database-facing model definitions
- `template_modules.validator.ts`: request validation schemas
- `template_modules.service.ts`: business logic

Các file cần có:

- `template_modules.route.ts`: khai báo route
- `template_modules.type.ts`: kiểu dữ liệu TypeScript
- `template_modules.controller.ts`: hàm controller xử lý HTTP
- `template_modules.model.ts`: định nghĩa model làm việc với database
- `template_modules.validator.ts`: schema validate request
- `template_modules.service.ts`: business logic
