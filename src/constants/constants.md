# constants

English: This folder contains shared constants used across the API, such as HTTP status codes, reusable enum-like values, and fixed application settings.

Tiếng Việt: Folder này chứa các hằng số dùng chung trong API, ví dụ HTTP status code, các giá trị dạng enum tái sử dụng và thiết lập cố định của ứng dụng.
