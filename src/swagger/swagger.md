# swagger

English: This folder contains Swagger runtime setup and optional OpenAPI document files. Use it to serve API documentation and store shared documentation fragments when needed.

Tiếng Việt: Folder này chứa phần setup Swagger khi chạy ứng dụng và các file tài liệu OpenAPI nếu cần. Dùng để phục vụ API documentation và lưu các phần tài liệu dùng chung.
