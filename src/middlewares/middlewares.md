# middlewares

English: This folder contains Express middlewares that run before or after route handlers. Use it for error handling, validation wrappers, authentication checks, request logging, and shared request processing.

Tiếng Việt: Folder này chứa Express middleware chạy trước hoặc sau route handler. Dùng cho xử lý lỗi, bọc validation, kiểm tra xác thực, ghi log request và xử lý request dùng chung.
