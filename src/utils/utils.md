# utils

English: This folder contains small reusable utility functions that are not tied to a specific feature module. Keep utilities generic and dependency-light.

Tiếng Việt: Folder này chứa các hàm tiện ích nhỏ có thể tái sử dụng và không phụ thuộc vào một module tính năng cụ thể. Nên giữ utility tổng quát và ít phụ thuộc.
